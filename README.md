Приложение, имитирующее логин - логаут на сервер, а также обработку некорректных данных, поступающих из АПИ.
Технологии:
- Многомодульность,
- Clean Architecture,
- Kotlin,
- Fragments,
- Retrofit,
- Coroutines.
